# 孔蓋中心座標計算器 (Circumcenter 3D)

一個純前端（HTML / CSS / JavaScript）的小工具，提供外業使用者輸入孔蓋邊框三點座標（北座標、東座標、高程），快速計算孔蓋中心座標（外心）與參考半徑，並顯示平面示意圖。

## 功能特色

- 輸入 A / B / C 三點的北座標、東座標、高程
- 計算 3D 外心（孔蓋中心）與參考半徑
- 支援在「北座標」欄一次貼上 Excel 三欄資料（北、東、高）
- 一鍵複製中心座標（Tab 分隔，可直接貼到 Excel）
- 顯示 ABC 平面投影示意圖（點位、外接圓、中心）
- 計算成功後自動捲動到結果區
- 右下角提供回到頂部按鈕

## 使用方式

1. 直接用瀏覽器開啟 `index.html`
2. 輸入（或貼上）A、B、C 三點座標
3. 點擊「計算孔蓋中心」
4. 在結果區查看中心座標與示意圖
5. 需要貼到 Excel 時可點「複製孔蓋中心（北/東/高）」

## 專案結構

- `index.html`：頁面結構
- `style.css`：版面與樣式
- `script.js`：計算邏輯與互動功能

## GitHub Pages 上架

此專案為純靜態頁面，可直接使用 GitHub Pages：

1. 進入 Repo `Settings` → `Pages`
2. `Source` 選 `Deploy from a branch`
3. Branch 選 `main`，資料夾選 `/ (root)`
4. 儲存後等待部署完成

部署後網址通常為：

`https://<your-username>.github.io/<repo-name>/`

## 注意事項

- 三點若幾乎共線，無法定義唯一外心，系統會提示錯誤
- 示意圖為 ABC 平面投影，供視覺檢核使用
